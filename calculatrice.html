Politique de Confidentialité - Calculatrice Turbo

Dernière mise à jour : 12 février 2026

Cette politique décrit comment l'application Calculatrice Turbo respecte la vie privée des utilisateurs. Nous attachons une grande importance à la protection de vos données personnelles.

  • L’application utilise le SDK AdMob pour diffuser des publicités, y compris des publicités récompensées pour débloquer certaines fonctionnalités comme le calculateur d’âge pendant 1 heure.
  • Les seules données collectées par AdMob sont les identifiants publicitaires (Advertising ID) pour diffuser et mesurer les publicités.
  • Aucune autre donnée personnelle n’est collectée ou transmise.
  • L’application peut enregistrer un historique des calculs et d'autres paramètres sur l’appareil dans une base de données SQLite.
  • Ces données ne sont jamais transmises à des serveurs externes ni partagées avec des tiers.
  • L’utilisateur peut supprimer ces données à tout moment via l’interface de l’application.
  • Les publicités peuvent être récompensées pour débloquer certaines fonctionnalités.
  • Les données utilisées par AdMob sont uniquement l’Advertising ID pour gérer les annonces.
  • Les utilisateurs peuvent gérer ou réinitialiser l’Advertising ID via les paramètres publicitaires de Google.

Toutes les données stockées localement sur l’appareil sont sécurisées. Les données utilisées par AdMob sont transférées via des connexions sécurisées (HTTPS).

  • Les utilisateurs peuvent supprimer toutes les données locales directement depuis l’application.
  • Les données collectées par AdMob (Advertising ID) peuvent être réinitialisées via les paramètres publicitaires Android.
  • Aucune donnée personnelle n’est collectée, traitée ou transmise par l’application en dehors d’AdMob.
  • Toutes les données locales sont stockées sur l’appareil et peuvent être supprimées par l’utilisateur.

L’utilisateur peut consulter, modifier ou supprimer toutes les données locales à tout moment. Les données utilisées par AdMob peuvent être réinitialisées ou supprimées via les paramètres publicitaires Android.

Cette politique peut être mise à jour à tout moment. Les modifications seront publiées sur cette page.
