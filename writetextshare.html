Politique de Confidentialité

Application : Write Text Share

Dernière mise à jour : 26/02/2026

1. Introduction

Write Text Share est une application permettant d’écrire, modifier, importer et partager des textes dans différents formats (PDF ou formats personnalisés). Nous nous engageons à protéger votre vie privée et à respecter les réglementations internationales, notamment le Règlement Général sur la Protection des Données (RGPD).

2. Données collectées

Données locales

  • Textes créés, modifiés ou importés
  • Fichiers exportés (PDF ou autres extensions choisies)
  • Préférences d’utilisation

Ces données sont stockées uniquement sur votre appareil.

Données publicitaires et techniques

L’application utilise Google AdMob pour les publicités récompensées. AdMob peut collecter :

  • Identifiant publicitaire
  • Données techniques de l’appareil
  • Adresse IP (anonymisée si requis par la loi)
  • Données d’interaction avec les publicités

Consultez la politique Google : https://policies.google.com/privacy

3. Utilisation d’Internet

L’application nécessite une connexion Internet pour :

  • Afficher les publicités récompensées
  • Vérifier l’éligibilité pour déblocage temporaire des fonctionnalités premium
  • Importer certains modèles de textes en ligne (si disponible)

Les fonctionnalités principales restent accessibles hors ligne.

4. Publicité Récompensée

Les utilisateurs peuvent choisir volontairement de regarder une publicité afin de débloquer temporairement :

  • Importation avancée de templates
  • Exportation PDF
  • Exportation multi-format personnalisée
  • Accès complet aux fonctionnalités premium pendant une durée limitée

Aucune publicité n’est imposée pour l’utilisation de base.

5. Conformité RGPD et Gestion du Consentement (EEE)

Pour les utilisateurs situés dans l’Espace Économique Européen (EEE), au Royaume-Uni et en Suisse, Write Text Share respecte le Règlement Général sur la Protection des Données (RGPD).

Données susceptibles d’être collectées par Google AdMob

L’application utilise Google AdMob, un service fourni par Google LLC, pour afficher des publicités récompensées. AdMob peut collecter et traiter les données suivantes :

  • Identifiant publicitaire (Advertising ID)
  • Adresse IP (pouvant être anonymisée selon la région)
  • Données relatives à l’appareil (modèle, système, langue)
  • Données d’interaction avec les annonces
  • Données approximatives de localisation

Ces données sont collectées et traitées directement par Google conformément à sa politique de confidentialité :

https://policies.google.com/privacy

Informations spécifiques à la publicité :

https://support.google.com/admob/answer/6128543

Gestion du consentement (CMP / Google UMP)

Retirer ou modifier votre consentement

Vous pouvez :

  • Réinitialiser ou supprimer votre identifiant publicitaire dans les paramètres Android
  • Modifier vos préférences publicitaires ici :

https://adssettings.google.com

Demande de suppression des données

Write Text Share ne stocke aucune donnée personnelle sur ses propres serveurs. Les données publicitaires sont traitées par Google.

Pour demander la suppression des données collectées par Google, vous pouvez utiliser :

https://support.google.com/policies/troubleshooter/7575787

6. Sécurité des données

Les données locales sont protégées par les mécanismes de sécurité Android. Aucune base de données distante n’est utilisée pour stocker vos textes personnels.

7. Conservation et Suppression des Données

Conservation des données locales

Les textes créés, importés ou modifiés dans l’application sont stockés exclusivement sur votre appareil (base de données locale). Ces données sont conservées tant que vous utilisez l’application ou jusqu’à leur suppression volontaire.

Suppression individuelle

L’application met à disposition des boutons permettant :

  • La suppression individuelle d’un texte spécifique
  • La suppression sélective de contenus enregistrés

La suppression est immédiate et irréversible.

Suppression complète des données

Un bouton dédié permet également de supprimer l’ensemble des données locales de l’application en une seule action (suppression totale des textes et contenus enregistrés).

Désinstallation de l’application

La désinstallation de Write Text Share entraîne automatiquement la suppression de toutes les données stockées localement sur votre appareil.

Données publicitaires (AdMob)

Les données collectées par Google AdMob ne sont pas stockées par l’application elle-même. Elles sont traitées directement par Google conformément à sa politique de confidentialité.

Pour demander la suppression ou gérer les données publicitaires, vous pouvez consulter :

https://adssettings.google.com

ou exercer vos droits via :

https://support.google.com/policies/troubleshooter/7575787

8. Modifications

Cette politique peut être mise à jour en cas d’évolution des fonctionnalités ou des services publicitaires.
